# Pipedrive Contact Panel

This is a Next.js (App Router + TypeScript + Tailwind) starter for embedding a contact panel in Pipedrive.

Quick start:

```bash
npm install
npm run dev
```

Open http://localhost:3000/panel to view the panel.

Remember to update `public/app-extensions.json` with your deployed URL before adding the extension manifest in Pipedrive.
